# 📝 To-Do App with Local Storage (Html,Css,JavaScript)

## 🚀 Project Overview
This is a simple and responsive To-Do App built using HTML, CSS, and JavaScript.  
It allows users to manage daily tasks efficiently with persistent storage.

---

## ✨ Features
- ➕ Add new tasks  
- ❌ Delete tasks  
- ✅ Mark tasks as completed  
- 💾 Saves data using Local Storage  
- 📱 Responsive design  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
